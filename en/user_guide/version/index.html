Version

SLYR is continually updated as a response to: - User requests: we listen and act so make sure you contact us if you come across any bugs or you have an idea. - Software development: as QGIS and ESRI continully develop their programs, we endeavour to include any new updates or changes in SLYR. If you think we have missed something, please contact us.
